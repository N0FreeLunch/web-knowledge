## XMLHttpRequest 
- 자바스크립트에서 XMLHttpRequest 객체를 호출하여 HTTP 통신을 사용할 수 있다.
- 최근에는 Fetch API를 사용하여 HTTP 통신을 할 수 있다.

## 리퀘스트 없는 통신
- 리퀘스트 없이 서버에서 클라이언트로 단방향 통신을 할 수 있는 방법이다.
